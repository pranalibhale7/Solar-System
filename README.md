# Solar System Website (HTML & CSS)

This is a **basic frontend project** created to understand and practice the core concepts of **HTML5 and CSS3**.

The project visually represents the **Solar System**, including planets, dwarf planets, and a simple orbit simulation, using only HTML and CSS.

---

## Purpose of the Project

This project was built as a **learning exercise** to:
- Understand HTML page structure
- Practice CSS styling, layouts, and animations
- Learn how multi-page websites work
- Get comfortable with linking files, images, and stylesheets

It does **not** use JavaScript or any external frameworks.

---

## Tech Stack

- **HTML5** – Page structure and content  
- **CSS3** – Styling, layouts, animations, and effects  

---

## Project Features

- Home page introducing the Solar System
- Separate pages for:
  - The Sun
  - Planets
  - Dwarf Planets
- CSS-based Solar System simulation
- Simple feedback/review form
- Multi-page navigation
- Static frontend (no backend)

---

## How to Access the Frontend - Run Locally
1. Download or clone the repository
2. Open `index.html` in any web browser
3. Navigate using the menu links

## Notes
